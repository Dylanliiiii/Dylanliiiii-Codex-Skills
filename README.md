# Dylanliiiii Codex Skills

这个仓库用于整理、保存和记录我自己创建或维护过的 Codex Skills。

## Skills

| Skill | 用途 |
| --- | --- |
| `sync-github-to-cnb` | 记录如何把 GitHub 仓库、Git tag、Release 附件和国内更新源同步打通到 CNB，适合需要国内访问通道的项目。 |
| `sync-personal-skills-repo` | 记录个人 Codex Skills 仓库的半自动同步流程：新增或修改 Skill 后，校验、更新 README，并在确认后提交推送。 |
